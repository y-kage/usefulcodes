# Mode
Finish modes by \<ESC\>
- input mode
  ```
  i
  ```
- command mode
  ```
  <ESC>
  ```
- visual mode
  ```
  v
  ```

# Finish
Use commands at command mode
- Save and Finish
  ```
  :wq
  ZZ
  ```
- Discard and Finish
  ```
  :q!
  ```

# Copy & Paste
- Copy
  ```
  y
  ```
- Cut
  ```
  d
  ```
- Paste
  ```
  p
  ```
- Select Range
  1. Move cursor to start point
  2. Press \<v\> and start visual mode
  3. Move cursor to end point

# Move cursor
- arrow keys
- alphabet keys
  h : left
  j : down
  k : up
  l : right
